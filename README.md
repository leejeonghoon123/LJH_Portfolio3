### 1️⃣ 전자계약 시스템 구축 및 유지보수

- **기간**: 2023.08 ~ 2024.11  
- **기술스택**: Java, 전자정부프레임워크, MySQL  
- **연동 API**:
  - 카카오 알림톡 전송: **비즈톡**
  - 본인인증: **Dream Security**
  - 메일 전송: **Daum SMTP**
- **역할**:
  - 배포 및 운영 환경 유지보수  
  - 외부 API 연동 로직 점검 및 수정  
  - 사용자 전자계약 처리 흐름 개선

### 2️⃣ 연말정산 시스템 유지보수 및 법 개정 반영

- **기술스택**: Java, MySQL / MSSQL / Oracle / PostgreSQL

### 3️⃣ 급여 시스템 고도화 및 전반 개발
